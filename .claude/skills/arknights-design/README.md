## 触发方式

以下任一方式均可激活本 skill：

- 输入 slash 命令：`/arknights-design`
- 自然语言描述，例如：
  - "用方舟风格帮我生成一个页面"
  - "做一个炎国风的展示页"
  - "用莱茵生命的配色写个博客模板"
  - "需要一个深色工业科幻风格的 HTML"

## 快速匹配

无需指定故事线时，系统按关键词自动匹配色板：

| 关键词 | 匹配色板 | 主色 |
|-------|---------|------|
| 科技、代码、研发、实验室 | 莱茵生命 | `#40b4c0` |
| 古典、欧洲、骑士、贵族 | 维多利亚 / 卡西米尔 | `#b89a50` / `#d4a030` |
| 中国风、国风、春节、诗词 | 炎国 | `#c04040` |
| 黑暗、恐怖、深海、未知 | 深海 | `#3a6a9a` |
| 夏天、复古、怀旧、音乐 | 火山旅梦 | `#e8a850` |
| 宗教、神圣、教堂、天使 | 拉特兰 | `#d4b896` |
| 黑帮、犯罪、意大利 | 叙拉古 | `#c04040` |
| 音乐、艺术、古典乐、歌剧 | 莱塔尼亚 | `#a060c0` |
| 未指定 | 罗德岛通用（默认） | `#f0a030` |

## 工作流程

1. 用户描述需求 → 匹配故事线色板
2. 按 `SKILL.md` 中的组件模式拼装页面结构
3. 从 `base.css` 复制样式，替换 `--accent` 变量
4. 输出纯 HTML 文件，所有 CSS 内联

## 文件说明

```
arknights-design/
├── README.md   ← 本文件
├── SKILL.md    ← 设计系统完整引用指南（色板、组件、铁律）
└── base.css    ← 完整 CSS 模板，可直接复制使用
```
