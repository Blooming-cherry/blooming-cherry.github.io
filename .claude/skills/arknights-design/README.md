# Arknights Design — 明日方舟设计系统

基于《明日方舟》UI/美术语言提炼的可复用 HTML/CSS/JS 设计系统。

## 触发方式

- 输入 `/arknights-design`
- 自然语言："用方舟风格生成一个页面"、"做一个炎国风的展示页"

## 功能概览

- **14条故事线完整色板** — 罗德岛、龙门、维多利亚、卡西米尔、莱茵生命、炎国、拉特兰、叙拉古、伊比利亚、谢拉格、多索雷斯、火山旅梦、莱塔尼亚 + 默认通用
- **动态 UI** — 陀螺仪视差、浮动粒子 canvas、鼠标辉光、滚动揭示（全部内联，零依赖）
- **丰富纹理** — 毛玻璃、噪点、CRT扫描线、半调网点、警戒线、故障抖动
- **响应式** — 移动端自适应

## 文件

```
arknights-design/
├── README.md   ← 本文件
├── SKILL.md    ← 完整设计参考（色板、组件、动画、铁律）
└── base.css    ← 完整 CSS 模板，可直接内联使用
```

## 快速开始

1. 从 `base.css` 复制全部 CSS 到 `<style>` 标签
2. 在 `:root` 中取消注释目标故事线的变量块
3. 从 `SKILL.md` 选取组件 HTML 结构
4. 从 `SKILL.md` 复制动态 JS 到 `</body>` 前
