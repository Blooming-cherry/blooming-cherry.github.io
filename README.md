# My-Hexo-Blog
# 我的博客

这是我的个人博客项目，基于 [Hexo](https://hexo.io/) 搭建。

## 功能简介
- 使用 Hexo 静态博客框架
- 支持 Markdown 写作
- 部署到 GitHub Pages
- 可扩展主题与插件

## 安装步骤
1. 克隆仓库：
   ###bash
   git clone https://github.com/Blooming-cherry/My-Hexo-Blog.git
2. 安装依赖：
   ###Bash
   npm install
3. 本地运行：
   ###Bash
   hexo  server
   浏览器访问 http://localhoxst:4000
## 部署方法
1. 使用 hexo generate 生成静态文件
2. 使用 hexo deploy 部署到 GitHub Pages
