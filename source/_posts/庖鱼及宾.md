# 夏意正盎

## ping网的波折
起初，我们按照gtp的建议，先安装了base-devel开发工具包组和git，构建基础的开发环境
```bash
sudo pacman -S base-devel git
```
但是我马上就遇到问题了![网络报错](D:\my-blog\my-blog\source\images\4.blog\2.png)在这个页面中，最初的报错是镜像源中未查找到我尝试安装的包，继而我尝试进行连接百度的网址，也就是ping网，然后出现了访问不到网络的情况。经过初步排查，是虚拟机的网络设置问题。这里我采用systed-netwokd的网络配置方式，配置文件使用[Match]和[Network]段，具体bash命令如下：
```bash
ip link
```
然后获得我的网络接口为[ens33]然后启用systermd-network，终端命令如下：
```bash
 #命令1
 systemctl enable --now systemd-networkd
 #命令2
 systemctl enable --now systemd-resolved
 ```
 然后配置DHCP，创建系统文件[/etc/systemd/network/20-wired.network]*此处文本命名仅作为示例*，配置文件如下：
 ```bash
 [Match]
 Name=ens33
 [Network]
 DHCP=yes
 #---<然后配置静态IP>---
Address=192.168.1.1
Gateway=192.168.1.1
DNS=8.8.8.8
DNS=1.1.1.1
#---<以上均作为示例，不代表实际配置>---
```
