## C++链表
在熟悉链表的操作之后我就开始学习交换节点，这个就引入了如何借助指针来删除节点，同时释放内存。假设我们要删除第N个节点，那么我们会有如下代码：
```Cpp
class solution{
	public:
		ListNode* (int val，ListNode* next)
		{
			int val;
			ListNode* next;
			ListNode(int val,ListNode* next) : val(val)  ; next(next){}
		};
		ListNode* remove(ListNode* head,int N)
		{
			ListNode* dummy = new ListNode(0);
			dummy->next = head;
			ListNode* fast = head;
			ListNode* slow = head;
			
			while (fast->next != nullptr && N != 0)
			{
				fast = fast->next;
				N--;
			}
			while (fast != nullptr)
			{
				slow = slow->next;
				fast = fast->next;
			}
			ListNode* tmp = slow->next;
			slow->next = slow->next->next;
			delete tmp;
			ListNode* result = dummy->next;
			delete dummy;
			return result;
		}
};
```
其中涉及了快慢指针和释放内存的基本操作和一些数学上的逻辑，总体而言是非常简单的（真的吗？真的吗🥹？我的小猪脑🫪一整个要烧了。。。）。
