# 夏意正盎
五月十日下午，阳光正好，青纱漫卷，一帘夏至，云卷云舒，花鸟缱绻，学生倥偬驹过隙，诗眼倦天涯。从上午十点开始便浸淫在archlinux，急头白脸地上B站选择自己的桌面环境，然后阅读官网文章，学习搭建桌面环境，然后查看archwiki，学习pacman（然而现在看来并无什么用，接近于无用功）的命令，然后指望从里面看出自定义桌面环境的搭建方法。
事实上当我发现，我的vmtools都没有办法正常使用之后，我已经lose my temper了，随后百无聊赖地打开桌面上的steam查看自己的库（实际上只有Apex和CS2），准备开始用FPS麻痹一下自己纤弱的神经。直到我收到这条信息![聊天记录](/attachments/庖鱼及宾.blog/1.png)，本来抱着试一试的心态，邀请他来帮我进行桌面环境安装，但是由于是第一次上手archlinux，也是一波三折。
## ping网的波折
起初，我们按照gtp的建议，先安装了base-devel开发工具包组和git，构建基础的开发环境
```bash
sudo pacman -S base-devel git
```
但是我马上就遇到问题了![网络报错](/attachments/庖鱼及宾.blog/2.png)在这个页面中，最初的报错是镜像源中未查找到我尝试安装的包，继而我尝试进行连接百度的网址，也就是ping网址，然后出现了访问不到网络的情况。经过初步排查，是虚拟机的网络设置问题。这里我采用systemd-netwokd的网络配置方式，配置文件使用[Match]和[Network]段，具体bash命令如下：
```bash
ip link
```
然后获得我的网络接口为[ens33]然后启用systermd-network，终端命令如下：
```bash
 #命令1
 systemctl enable --now systemd-networkd
 #命令2
 systemctl enable --now systemd-resolved
 ```
 然后配置DHCP，创建系统文件[/etc/systemd/network/20-wired.network]*此处文本命名仅作为示例*，配置文件如下：
 ```bash
 [Match]
 Name=ens33
 [Network]
 DHCP=yes
 #---<然后配置静态IP>---
Address=192.168.1.1
Gateway=192.168.1.1
DNS=8.8.8.8
DNS=1.1.1.1
#---<以上均作为示例，不代表实际配置>---
```
配置完临时网络连接可谓是轻舟已过万重山，没想到出去之后又是泰坦尼克大冰山。
## 一波未平一波又起
尝试安装Open-VM-Tools绝对又是一个灾难，由于没有经验，当我使用
```bash
sudo pacman -S Open-VM-Tools
```
这个命令的时候，即使我使用了
```bash
sudo pacman -Qs Open-VM-Tolls
```
检查到本地安装包，明明全部都下载下来了但是我仍然无法共用主机的剪贴板，因此产生了不少麻烦事情到这里我暂且放弃了，手动下载了python（下载之后的archlinux的库文件均省略指令），因此，我产生了使用python程序的想法。这时非常恰巧，我那段时间正在摆弄一个基于claudecode源代码的python改写版agent，在github上的项目名称为[OpenHarness](https://github.com/HKUDS/OpenHarness.git)*此链接可以直接克隆*下面感兴趣的可以参见这个[知乎文章](https://zhuanlan.zhihu.com/p/2023517045422047369)。
在其中，还发生了一些小插曲：
*1.*我在下载openharness的时候， 因为没有配置网络代理，从而![无法识别我的git指令](/attachments/庖鱼及宾.blog/5.png)，随后顺利地下载下来。
*2.*虽然下载了openharness，但是仍然无法解决我的vmtools的下载问题，于是处在悬而未决的状态，留了下来。
*3.*然后，我按照正常的安装流程，尝试下载fcitx5中文输入法，但是出现了渲染问题，所有待选词成为了“豆腐块”。
对于第一个问题，我很快地解决了，但是对于后两个，我在wiki上翻来覆去，百思不得其解。于是我在大概5月24日的时候，我向axty佬申请技术外援，给我远程fix一下。在一个小时的时间里，后两个问题迎刃而解。
对于第二个问题，参见[wiki](https://wiki.archlinuxcn.org/wiki/VMware/%E5%AE%89%E8%A3%85_Arch_Linux_%E4%B8%BA%E8%99%9A%E6%8B%9F%E6%9C%BA)，主要问题出在![7.6.2](/attachments/庖鱼及宾.blog/6.jpeg)，而且恰好我的桌面使用的是cosmic，在部分功能上仍然需要安装这些包。
对于第三个问题说来搞笑，在我已经下载了完整的fcitx5的安装包之后，仍然显示豆腐块，原因是其默认字形是![日语字形](/attachments/庖鱼及宾.blog/3.png)，发现问题和解决问题的 [帖子来自](https://szclsya.me/zh-cn/posts/fonts/linux-config-guide/#footnote-1)。以上就是我在配置archlinux时候的波折了。
## 结语
中间搁笔了许久，合肥市的暴雨来去匆匆，太阳的笑靥初绽，晴日方好。我曾向往潜公笔下的“晨兴理荒秽，带月荷锄归”，也曾神往唐寅笔下的“酒醒只在花前坐，酒醉还来花下眠”，如今，一切都在眼前徐徐展开。六月，是生命盛放的月份“玉梯凝望久，叹芳草萋萋千里“，接过姜夔的汉酺，轻轻一泻，酒的醇香溶溶渫渫，怀中升起“此地，宜有词仙，拥素云黄鹤，与君游戏”的激荡。原来，洛阳花不晚，长安夜未央。
