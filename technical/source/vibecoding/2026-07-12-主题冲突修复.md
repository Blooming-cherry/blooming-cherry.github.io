## 问题

嵌入 Hexo 的 C 链表学习笔记页面存在三类冲突：SVG 子元素被 markdown 渲染器转义为 `&lt;text>` 导致图表破碎；`canvas` 裸选择器覆盖鼠标拖尾尺寸；`*` / `html` / `body` 等全局 CSS 规则破坏 NexT 主题布局。

## 解决

1. 源文件从 `.md` 改为 `.html`，绕过 Hexo markdown 转义
2. `canvas { height: 320px }` → `.canvas-box canvas`，仅作用于演示区
3. 删除 `*`、`html`、`body`、`body::before`、`main` 五条全局选择器
