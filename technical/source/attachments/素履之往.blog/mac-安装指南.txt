====================================================================
      MacBook 全套环境安装指南（网络受限版）
      (Ollama + Open WebUI + Qwen2.5 14b + Homebrew)
====================================================================

本文件夹内容：
  📦 Ollama.dmg                      — Ollama Mac 安装包（172MB）
  📦 Docker-AppleSilicon.dmg         — Docker Desktop Mac 版（542MB）
  📁 homebrew-install/               — Homebrew 安装脚本（可离线执行！）
  📄 本文件                           — 照着做就行


====================================================================
【第 1 步：安装 Homebrew】（本地安装，不需要网络）
====================================================================

把整个 mac-setup 文件夹拷到 Mac 上（比如放到 ~/Downloads/mac-setup）。

打开 Mac「终端」App，运行：

  # 进入本文件夹
  cd ~/Downloads/mac-setup

  # 本地安装 Homebrew（不需要网络）
  /bin/bash homebrew-install/install.sh

⚠️ 装完后终端会提示你运行 PATH 设置命令，一定要跑，类似：

  echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
  eval "$(/opt/homebrew/bin/brew shellenv)"

验证：
  brew --version


🌟 Homebrew 首次使用前，配一下清华镜像（加速后续 brew install）：

  # 替换 Homebrew 源
  export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
  export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
  export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"

  # 写入配置文件，永久生效
  echo 'export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"' >> ~/.zshrc
  echo 'export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"' >> ~/.zshrc
  echo 'export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"' >> ~/.zshrc
  source ~/.zshrc


====================================================================
【第 2 步：安装 Ollama】
====================================================================

双击本目录下的 Ollama.dmg → 拖进 Applications → 双击启动即可。

或者用命令行：
  open Ollama.dmg
  # 然后把 Ollama.app 拖进 /Applications

验证：
  ollama --version


====================================================================
【第 3 步：拉取 Qwen2.5 14b 模型】
====================================================================

确定 Ollama 在运行（菜单栏有 🦙 图标），然后终端运行：

  ollama pull qwen2.5:14b

模型约 9GB，需要网络下载。如果网络慢，清华镜像可能帮不上
Ollama 的忙，可以考虑：
  - 换手机热点下载
  - 换成 7b（ollama pull qwen2.5:7b，只有 4.7GB）

验证：
  ollama list
  ollama run qwen2.5:14b


====================================================================
【第 4 步：安装 Docker + Open WebUI】
====================================================================

4a. 装 Docker Desktop：
    双击本目录下的 Docker-AppleSilicon.dmg
    拖进 Applications → 双击启动 → 完成初始化引导
    （首次启动会要求输入 Mac 密码，正常）

4b. 拉取并启动 Open WebUI（需要网络下载镜像，约 1.5GB）：

    docker pull ghcr.io/open-webui/open-webui:main

    docker run -d -p 3000:8080 \
      -v ~/.ollama:/root/.ollama \
      -v open-webui:/app/backend/data \
      --name open-webui \
      --restart always \
      ghcr.io/open-webui/open-webui:main

4c. 浏览器打开：
    http://localhost:3000

    首次进入后，选择 Ollama 作为后端即可使用。


====================================================================
【各组件占用】
====================================================================

              磁盘占用      运行时 RAM
Homebrew       ~0.5 GB        —
Ollama         ~0.3 GB        ~0.2 GB
Qwen2.5 14b    ~9.0 GB        ~10 GB
Docker         ~2.0 GB        ~1 GB
Open WebUI     ~1.5 GB        ~0.3 GB
─────────────────────────────────────
合计          ~13.3 GB        ~11.5 GB

你的 MacBook：16GB 统一内存，可用约 13GB。
14b 模型吃 ~10GB → 刚好在极限，建议关掉不必要的应用。
如果感觉卡，换 qwen2.5:7b（只吃约 6GB，非常舒服）。
